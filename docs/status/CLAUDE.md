# docs/status/
Live execution state and handoff docs.

## Files
- `PROJECT-STATUS.md`: latest shipped state and validation snapshot
- `FEATURE-MATRIX.md`: tracked features and their current status
- `CLAUDE-HANDOFF.md`: concise handoff for the next coding pass

## Convention
- Update `PROJECT-STATUS.md` and `FEATURE-MATRIX.md` together.
- Keep date stamps in KST.
